기능

1. 게임을 위한 숫자 선택
- 1 ~ 9까지의 수로 중복 없이 숫자 3개 고르기

2. 사용자 입력을 받음 (반복)
- User Input 받기 (readLine 사용)
- Input 유효성 확인
  (길이가 3인가?)
  (숫자인가?)
  (잘못된 값이면 IllegalArgumentException발생 + 종료)
- 사용자 입력값의 형을 변환
- 컴퓨터가 고른 무작위 숫자와 사용자 입력을 비교
- 자리까지 맞춤 -> strike ++
- 번호는 있음 -> ball ++
- 3스트라이크 나올 때까지 반복
- 3스트라이크 이후 다시 할지 여부를 물어봄 ( 1, 2 )
